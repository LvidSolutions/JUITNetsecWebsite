# JUIT NetSec AB

Grund för en modern single-page website för JUIT NetSec AB.

## Teknik

- React
- Vite
- Tailwind CSS
- Komponentbaserad struktur
- Responsiv layout för mobil, tablet och desktop

## Kom igång

```bash
npm install
npm run dev
```

## Struktur

- `src/components/ui` innehåller återanvändbara grundkomponenter.
- `src/styles/index.css` innehåller global styling och Tailwind-lager.
- `tailwind.config.js` innehåller färger, typografi och grundtokens.
